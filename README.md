З директорії server
* Сервер запускається командою
``` bash 
 go run ./cmd/server
```
З директорії client
* Приклади реалізації сценаріїв на базі клієнтського SDK запускаються через команду

```
npm install
node examples.js
```
* Папка `db` містить SQL скрипти для ініціалізації БД сервера.
